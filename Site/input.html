Input

The Input System in Chai is very rudimentary and very simple. All you can do for now is bind keyboard inputs to specific name-tags, and check for their pressing behaviour. Additionally, you can check mouse and touch events.

func BindInput(_input_name string, _corr_key KeyCode)

// Usage
chai.BindInput("Jump", chai.KEY_SPACE)

_input_name: This input's name-tag.

_corr_key: The corresponding keycode for this input.

package main

import (
	chai "github.com/mhamedGd/chai"
)

var app chai.App = chai.App {
    Title: "",
    Width: 1920,
    Height: 1080,
    OnStart: func(){
        chai.BindInput("Jump", chai.KEY_SPACE)
    },
    OnUpdate: func(_delta_time float32){
        if (chai.IsJustPressed("Jump")) {
            // Prints a Message to the Console
            // To see the console, in the browser press (CTRL/CMD + L Shift + C)
            chai.LogF("Just Pressed Jump!")
        }
    },
    OnDraw: func(_delta_time float32){},
    OnEvent: func(_event *chai.AppEvent){},
}

func main() {
    chai.Run(&app)
}

Input Methods

func ChangeInputBinding(_input_name string, _new_binding KeyCode)

_input_name: This input's name-tag.

_new_binding: The corresponding new keycode for this input.

Changes the Keycode of an already bound Input.


func GetActionStrength(_input_name string) float32

_input_name: The name-tag of the Input you're trying to check.

Returns 1.0 of Input is pressed.


func IsPressed(_input_name string) bool

_input_name: The name-tag of the Input you're trying to check.

Returns true if Input is pressed.


func IsJustPressed(_input_name string) bool

_input_name: The name-tag of the Input you're trying to check.

Returns true if Input was just pressed.


func IsJustReleased(_input_name string) bool

_input_name: The name-tag of the Input you're trying to check.

Returns true if Input was just released.


func IsMousePressed(_mouse_button MouseButton) bool

_mouse_button: Current Mouse Button being pressed

  • chai.LEFT_MOUSE_BUTTON
  • chai.MIDLE_MOUSE_BUTTON
  • chai.RIGHT_MOUSE_BUTTON

Returns true if mouse button is pressed.


func IsMouseJustPressed() bool

Returns true if a mouse button was just pressed.


func IsMouseJustReleased() bool

Returns true if a mouse button was just pressed.


func GetNumberOfFingersTouching() uint8

Returns the current number of fingers touching.


func AreFingersTouching(_num_of_fingers uint8) bool

Returns true if the number provided is equal to the number of fingers touching.


func IsJustTouched(_num_of_fingers uint8) bool

Returns true is this number of fingers has just touched the screen.


func IsJustTouchReleased(_num_of_fingers uint8) bool

Returns true is this number of fingers were just touching the screen before the user took a finger off of it.
